多线程相关笔记

1: 为什么需要并行（多线程）
			1. 业务需求
		
		 eg: 一个http服务器同一时刻需要去处理多个客户端的请求。 通常的做法是为每个客户端去开启一个线程。
		
		2. 性能需求

			由于硬件的原因，芯片发展的计算速度限制（4Hz）,我们为了性能而折中使用的是一个cpu中插入多个芯片（多核）
